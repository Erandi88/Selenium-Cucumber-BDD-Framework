# Selenium Cucumber BDD Framework

## 📖 Project Overview

This project is a Behaviour-Driven Development (BDD) automation framework built using Java, Selenium WebDriver, and Cucumber.
It is designed to automate web application testing in a simple and maintainable way.

## 🛠️ Technologies Used

* Java
* Selenium WebDriver
* Cucumber (BDD)
* Maven

## 🚀 Features

* BDD-style test scenarios using Gherkin
* Reusable step definitions
* Easy test execution using Maven

## ▶️ How to Run the Project

1. Clone the repository
2. Open the project in IntelliJ or Eclipse
3. Run tests using:
   mvn test

## 📂 Project Structure

* `features` → Cucumber feature files
* `stepdefinitions` → Step definition classes
* `pages` → Page Object Model classes
* `utils` → Utility classes
